# いかりスーパー周辺店舗マップ (独立版)

このプロジェクトは、いかりスーパーの全店舗の位置関係と店舗情報を地図上にプロットして表示する、自己完結した独立型Webマップツールです。

## 構成ファイル

プロジェクトフォルダ内のファイルは、役割に応じて明確に3レイヤーに分離して管理されています。

- **`stores_db_base.json` (位置情報マスタ - 開発者向け)**:
  各店舗の「店舗名」と「緯度経度 (`coords`)」のみを保持しています。マップ上のプロット位置を変更したい場合のみ、このファイルの数値を直接編集します。
- **`config.md` (店舗情報設定 - ユーザー向け)**:
  住所、電話番号、駐車場、営業時間、特徴、画像URLなどを一元管理する Markdown 形式のテーブルファイルです。手動で店舗情報をカスタマイズしたい場合は、このファイルの表を直接編集して上書き保存します。
- **`update_store_db.py` (自動データ取得スクリプト)**:
  いかりスーパー公式サイトから最新の店舗情報を取得し、`config.md` のテーブル情報を自動更新します。※ユーザーが手動で書き込んだ「特徴」などのデータは上書きされずに保持されます。
- **`generate_map.py` (マップ生成スクリプト)**:
  `stores_db_base.json` と `config.md` を読み込んで結合し、ユーザーのデスクトップに `ikari_map.html` を出力します。
- **`map_only_template.html` (HTMLテンプレート)**:
  出力される `ikari_map.html` の見た目やスクリプトを定義したテンプレートファイルです。
- **`run_update.bat` (更新実行用バッチファイル)**:
  ダブルクリックするだけで、上記の「自動データ取得」と「マップの再生成」を一貫して実行します。

---

## 使い方・カスタマイズ方法

### 1. 手動で店舗情報を変更したい場合
1. **`config.md` を編集する**:
   メモ帳やテキストエディタで `config.md` を開き、変更したい店舗の行（例: 営業時間の列や駐車場の列）を書き換えて保存します。
2. **マップを再生成する**:
   フォルダ内の **`run_update.bat`** をダブルクリックして実行します。
3. **結果を確認する**:
   デスクトップにある **`ikari_map.html`** をブラウザで開いて（すでに開いている場合はF5キーで更新）、変更した情報が反映されていることを確認します。

### 2. 店舗情報を公式サイトの最新情報に自動更新したい場合
1. フォルダ内の **`run_update.bat`** をダブルクリックして実行します。
2. 自動的に公式サイトから情報（画像や営業時間など）を取得し、`config.md` のテーブルを自動更新した上で、デスクトップの `ikari_map.html` を再生成します。
   ※ `config.md` 内で手動で書き換えた「特徴」の記述は、自動更新時にも消えずにそのまま保持されます。

### 3. マップ上のピン（プロット）の位置を変更したい場合
1. **`stores_db_base.json` を編集する**:
   テキストエディタで `stores_db_base.json` を開き、対象店舗の `"coords": [緯度, 経度]` の数値を変更して保存します。
2. **`run_update.bat`** をダブルクリックして実行し、マップを再生成します。
